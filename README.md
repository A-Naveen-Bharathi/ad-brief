A self-contained HTML tool for generating structured ad creative briefs across Google, Meta, LinkedIn, Reddit, and Microsoft/Bing.
Select your ad formats (Single Image, Carousel, Video, Document/Lead Gen, Search RSA, Extensions), choose platforms, set the number of creative sets, and export a formatted Excel brief with three sheets:

Content Team — every copy field with character limits, variant slots, live character counters, and over-limit warnings
Design Team — all required asset dimensions, file specs, and durations
Brief Notes — campaign goal, audience, offer, and additional context

No dependencies. No installation. No internet required. Open in any browser and export.
